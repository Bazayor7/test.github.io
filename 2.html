HTML - (от английского. HyperText Markup Language -- «язык гипертекстовой разметки») - стандазированный язык гипертекстовой разметки документов для просмотра веб страниц в браузере
HTML был создан в 2010 году 1 января.
Элементы HTML являются строительными блоками HTML страниц. HTML предоставляет средства для создание заголовков, абзацев, списков, ссылок, цытат и других элементов. Элементы HTML выделяются тегами, записанными с использованием угловых скобок. Такие теги, как и , напрямую вводят контент на страницу. Другие теги, такие как , окружают и оформляют текст внутри себя и могут включать другие теги в качестве подэлементов. Браузеры не отображают HTML-теги, но используют их для интерпретации содержимого страницы.
